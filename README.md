# 文本转像素工厂 · 信息板

> 一款轻量级的前端工具，将纯文本瞬间转化为像素风格的视觉信息板。

**在线地址**：[https://mdt.morenw.top](https://mdt.morenw.top)

---

## ✨ 功能特性

- **文本转换**：输入任意文本内容，一键转为像素工厂信息板
- **响应式设计**：适配桌面端与移动端，随时随地使用
- **风格还原**：完美复刻原游戏按键，输入框，滚动条等控件，使用原游戏素材

---

## 🛠 技术栈

| 层级 | 技术 |
|------|------|
| 框架 / 库 | 原生 JavaScript / 现代前端框架 |
| 样式 | CSS3 (像素风格设计) |
| 部署 | Vercel / 静态资源托管 |
| 构建工具 | Vite + npm |

> 项目采用纯前端架构，开箱即用。

---

## 🚀 本地运行

### 环境要求

- **Node.js** >= 20.0.0
- **npm** >= 8.0.0

### 安装与启动

1. **克隆仓库**
   ```bash
   git clone https://github.com/moren-w/mindustry-reader.git
   cd mindustry-reader

2. **本地运行与部署**：完整的安装和启动流程
   - `npm install` — 安装项目依赖
   - `npm run dev` — 启动开发服务器
