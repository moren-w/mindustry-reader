# 文本转像素工厂 · 信息板

> 一款轻量级的前端工具，将纯文本瞬间转化为像素风格的视觉信息板。

**在线地址**：[https://mdt.morenw.top](https://mdt.morenw.top)

---

## ✨ 功能特性

- **文本转像素**：输入任意文本内容，一键生成像素风格的信息板
- **纯前端架构**：无需后端服务，所有处理均在浏览器端完成，保障数据隐私
- **即时预览**：输入内容实时渲染，所见即所得
- **像素美学**：复古像素风视觉呈现，适合用于公告、留言、创意展示等场景
- **响应式设计**：适配桌面端与移动端，随时随地使用

---

## 🛠 技术栈

| 层级 | 技术 |
|------|------|
| 框架 / 库 | 原生 JavaScript / 现代前端框架 |
| 样式 | CSS3 (像素风格设计) |
| 渲染 | HTML5 Canvas / DOM 像素映射 |
| 部署 | Vercel / 静态资源托管 |
| 构建工具 | Node.js + npm |

> 项目采用纯前端架构，无服务器端依赖，开箱即用。
